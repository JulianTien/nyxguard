# API Summary

Main backend endpoints include:

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/user/profile`
- `GET /api/guardians`
- `POST /api/trips`
- `POST /api/chat`
- `POST /api/chat/proactive`

Detailed behavior, product terminology, and database notes are documented in the existing files under `docs/`.
